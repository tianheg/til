# Setup Jekyll Environment
Jekyll requires the following:

*   Ruby version **2.4.0** or higher | [rbenv](https://github.com/rbenv/rbenv)
*   RubyGems
*   GCC and Make

```sh
gem install jekyll bundler
jekyll new myblog
cd myblog
bundle exec jekyll serve
```
