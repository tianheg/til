# Introduction to Front End Coding

## Front-end Develop Tips

### Disable export default

Using `export default` will make it very easy to load modules with errors when you mix cjs+esm.

ref: <https://zhuanlan.zhihu.com/p/97737035>, <https://zhuanlan.zhihu.com/p/97335917>

### Use `map` instead of `if else`

Too much `if else` in the code is undoubtedly very unreadable.

```js
// Before optimization
function getRes (path) {
    if (path === '/') {
        return 'index'
    } else if (path === '/news') {
        return 'news'
    } else {
        return '404'
    }
}
// After optimization

const map = {
    '/': 'index',
    '/news': 'news'
}

function getRes (path) {
   return map[path] || '404'
}
```
