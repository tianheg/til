# 学习路径

1. freeCodeCamp's JavaScript Algorithms and Data Structures
2. <https://javascript.info/> or [MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)
3. 复习 HTML，CSS：[HTML-W3Schools](https://www.w3schools.com/html/)，[CSS-Tricks](https://css-tricks.com/)
4. 学习 DOM 和 DOM 操作：[MDN](https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model)
5. 学习写应用：[JavaScript 30](https://javascript30.com/)

注：**在学习框架之前，打好基础**。
