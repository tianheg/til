# Best Practices With [News Translation][1]

## 翻译

1. 错误修改（英文原文的）

## 校对

1. 不要随意修改译者的翻译，也不要一字一句地琢磨，只要能够表达出英文的原意就行
2. 把一些口语用词去掉，比如，“能”，“好”等等。技术文章应该简洁明了，没有一句没有用处的话
3. 错误修改（英文原文的，译者的）
4. 只需要 `Add single comment`，不必 `Start a review`

## 准备工作

1. 专业词汇表
2. Markdown 中英文文章排版规范 [tianheg/document-style-guide](https://github.com/tianheg/document-style-guide)
