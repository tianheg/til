# Change Default Package Manager

```bash
vim ~/.config/gatsby/config.json
```

Change:

```json
"cli": {
        "packageManager": "npm"
}
```

to:

```json
"cli": {
        "packageManager": "yarn"
}
```

[source](https://www.gatsbyjs.com/docs/reference/gatsby-cli/#how-to-change-your-default-package-manager-for-your-next-project)
