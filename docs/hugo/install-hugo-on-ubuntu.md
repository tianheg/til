# Install Hugo On Ubuntu

```bash
wget https://github.com/gohugoio/hugo/releases/download/v0.81.0/hugo_extended_0.81.0_Linux-64bit.deb
sudo apt-get install ./hugo_extended_0.81.0_Linux-64bit.deb
```
