# Adjust The Margin Of An Element

```css
.red-box {
    background-color: crimson;
    color: #fff;
    padding: 20px;
    margin: 20px;
}

.blue-box {
    background-color: blue;
    color: #fff;
    padding: 20px;
    margin: 10px;
}
```

An element's `margin` controls the amount of space between an element's `border` and surrounding elements.
