# Open And Use BIOS In VMware Workstation

Add one of these words:

1. `bios.forceSetupOnce = "TRUE"`
2. `bios.bootDelay = "xxxx"`(xxxx means milliseconds, maximum value is 10000 milliseconds, also is 10 seconds)
