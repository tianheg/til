# Install Nnn

```sh
$ wget https://github.com/jarun/nnn/releases/download/v4.0/nnn_4.0-1_ubuntu20.04.amd64.deb
$ sudo apt install ./nnn_4.0-1_ubuntu20.04.amd64.deb
```
