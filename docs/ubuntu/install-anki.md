# Install Anki

Install from [here](https://apps.ankiweb.net/#download)

Then exec these orders:

```bash
$ tar xjf Downloads/anki-2.1.42-linux.tar.bz2
$ cd anki-2.1.42-linux
$ sudo ./install.sh
```
