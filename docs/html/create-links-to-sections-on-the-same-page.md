# Create Links To Sections On The Same Page

```html

    <a href="#nav-bar" class="nav-link">HTML</a> |
    <a href="#nav-bar" class="nav-link">CSS</a> |
    <a href="#nav-bar" class="nav-link">JS</a>
</nav>
```
