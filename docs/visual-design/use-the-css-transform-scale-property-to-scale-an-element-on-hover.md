# Use The CSS Transform Scale Property To Scale An Element On Hover

Applying a transform to a `div` element will also affect any child elements contained in the div.

```css
div {
    width: 70%;
    height: 100px;
    margin:  50px auto;
    background: linear-gradient(
      53deg,
      #ccfffc,
      #ffcccf
    );
}

div:hover {
    transform: scale(1.1);
}
```
