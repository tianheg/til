Before HTML¶

  1. 很多电脑，尤其 Web 服务器，是大小写敏感的

  2. 浏览器、Web 服务器、编程语言对空格的处理并不一致，目前的做法：文件/文件夹用小写字母；空格用 -(hyphen) 代替

  3. Google 搜索把 - 视为单词分隔符，而对于 _(underscore) 却没有这样对待

ref:

  1. https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/Dealing_with_files
