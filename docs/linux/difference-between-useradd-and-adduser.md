# Difference Between `useradd` And `adduser`

`useradd` can be used for any Linux distribution, including Ubuntu, CentOS, RHEL, Debian, Fedora, and Arch Linux.

`useradd` is a low\-level utility, Debian and Ubuntu users can use the friendlier [adduser](https://linuxize.com/post/how-to-add-and-delete-users-on-ubuntu-18-04/) command instead.
